Project Code Overflow:
Replica of Stack Overflow

Code Overflow: Replica of Stack Overflow

♦ Ruby on Rails ♦ Redcarpet markdown rendering gem ♦ Devise gem for authentication ♦ Pundit gem for authorization ♦ Faker gem ♦ Stripe gem to charge for premium users.
Code Overflow allows users to collaborate on community sourced posts.
User Stories
- As a user, I want to sign up for a free account by providing a user name, password and email.
- As a user, I want to sign in and out of Code Overflow.
- As a user with a standard account, I want to create, read, update, and delete public posts.
- As a developer, I want to offer three user roles: admin, standard, or premium.
- As a developer, I want to seed the development database automatically with users and posts.
- As a user, I want to upgrade my account from a free to a paid plan.
- As a premium user, I want to create private posts.
- As a user, I want to edit wikis using Markdown syntax.
- As a premium user, I want to add and remove collaborators for my private posts.
